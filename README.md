**CS2 ShareCfg** — это легковесная и быстрая утилита на C++ для автоматического экспорта всех настроек Counter-Strike 2. Программа извлекает ваши конфигурационные файлы (бинды, настройки мыши, графику, звук, радар, прицел) и бережно упаковывает их в готовый ZIP-архив для удобной передачи другим игрокам.

## 🚀 Ключевые особенности

* **Автоматическое определение профиля:** Утилита считывает данные из реестра Windows для точного определения SteamID активного пользователя. Вам не нужно искать свои папки вручную.
* **Сбор оригинальных файлов:** Копируются исходные файлы движка Source 2 (`.vcfg` и `.txt`), что гарантирует 100% перенос всех параметров, включая те, которые не работают через консольную команду `exec`.
* **Готовый ZIP-архив:** Программа использует встроенные средства Windows (PowerShell) для моментальной упаковки файлов в `CS2_SharedConfig.zip`, сохраняя его прямо в вашу папку «Документы».
* **Инструкция в комплекте:** В архив автоматически добавляется файл `ИНСТРУКЦИЯ.txt` с подробными пошаговыми действиями для получателя.
* **Никаких лишних библиотек:** Написано на C++ (стандарт 17) с использованием Windows API. Работает быстро и не требует установки.

## 🛠️ Как это работает

1. Скачайте и запустите `CS2_ShareCfg.exe`.
2. Запустите Counter-Strike 2 (утилита находится в режиме ожидания процесса `cs2.exe`).
3. Как только игра запустится, программа соберет конфигурации и создаст архив.
4. Откройте папку **Документы** (`Documents`), заберите свежий `CS2_SharedConfig.zip` и отправьте его другу!

## ⚠️ Важное примечание для получателей

В архив включен файл `cs2_video.txt`. Это означает, что **настройки графики и разрешение экрана** у человека, который применит этот конфиг, изменятся на ваши. Получателю необходимо будет зайти в настройки графики CS2 и вернуть свое родное разрешение монитора после первого запуска.
